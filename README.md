# TinyApp

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly or goog.gl).

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
